Sistema de Gestión de Personal Hospitalario

Este proyecto implementa un sistema orientado a objetos para gestionar diferentes tipos de empleados en un hospital.  
Incluye herencia, abstracción, encapsulamiento y polimorfismo


Clases incluidas

 Clase base:
- EmpleadoHospital (abstracta)

 Clases hijas:
- MedicoGeneral
- EspecialistaMedico
- Enfermeria
- TecnicoApoyo

Cada clase implementa:
- Cálculo de salario mensual
- Rol profesional
- Método `atender()`
- Bonificaciones por antigüedad
